Welcome to the window shops! Here you can find aero glass css styles for your websites! To use these styles, download the SVG and put it in the same folder as your html. Put the css code in your css code. To make an element glassy give the element the "glass" class. For coloured glass, use the class "glass" followed by the colour (e.g for green glass set the class to "glassgreen")
